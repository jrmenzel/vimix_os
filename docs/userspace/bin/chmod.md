# chmod - Change file mode

Changes the [file](../../kernel/file_system/file.md) mode.

> chmod `mode` `file`

`mode` is given in octal notation. 

**Returns:**
- 0 on success
**Syscall:** [chmod](../../kernel/syscalls/chmod.md)

---
**Up:** [user space](../userspace.md)
