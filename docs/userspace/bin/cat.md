# cat - concatenate files

Concatenates given files and writes all to [stdout](../../misc/stdio.md).
If no file is provided, cat reads from [stdin](../../misc/stdio.md).

> cat `FILE0` `FILE1` ... `FILEn`

**Returns:**
- 0 on success
